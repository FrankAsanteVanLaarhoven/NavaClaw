# 🚀 InsightsAI Platform - Complete Application Preview

## Overview

InsightsAI is a comprehensive web crawling and data analysis platform that combines advanced web scraping capabilities with modern analytics and collaboration tools. The application features a beautiful dark theme with cyan and purple accents, glassmorphism effects, and smooth animations.

## 🔧 Technical Architecture

### Frontend Stack
- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Shadcn/ui** components
- **Lucide React** icons

### Backend Integration
- **Node.js Backend** (Port 3001) - Main API server
- **Python Crawler Backend** (Port 8000) - Crawling engine
- **API Bridge** - Seamless communication between services

### Key Components
- **Navigation** - Responsive header with dropdown menus
- **HeroWithGlobe** - Animated 3D globe component
- **AnalyticsDashboard** - Data visualization interface
- **CollaborationInterface** - Team collaboration tools
- **WorkspaceCrawler** - Integrated crawler interface
- **DataBricksNotebook** - Interactive notebook component
